> **Stack:** `Python` `MySQL` `Streamlit` `Power BI`

- Integrated MySQL backend for historical data storage; Streamlit for real-time user interaction
- Delivered actionable investment signals replicating real-world Business Analyst BI deliverables

---

### 🏃 Fitbit Calorie Burn Prediction & Workout Pattern Clustering
> **Stack:** `Python` `Scikit-learn` `Pandas` `Matplotlib`

- Built regression models to predict calorie burn from wearable sensor data

- Full ML pipeline: EDA → Feature Engineering → Model Training → Evaluation

---

### 📉 CryptoCast — Bitcoin Price Forecasting with Deep Learning
> **Stack:** `Python` `LSTM` `TensorFlow` `Keras` `Pandas`

- Built multi-horizon LSTM time-series forecasting models on historical Bitcoin OHLCV data
- Visualized forecast confidence bands to support data-driven investment decisions

---

### 🎬 IMDB Movie Recommendation System (NLP)
